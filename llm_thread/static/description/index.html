LLM Thread Module

This module provides advanced functionality for managing and interacting with threads in a Large Language Model (LLM) context.

Chat Thread Management

Effortlessly create, update, and manage chat threads for seamless communication and collaboration.

Message List

View and interact with messages in a thread, with support for rich text, attachments, and more.

Composer View

Input and send messages within a thread using an intuitive and responsive composer interface.

Configuration

After installation:

  1. Configure LLM providers in the Settings menu.
  2. Set up API credentials for each provider.
  3. Configure default models for different use cases.
  4. Grant appropriate access rights to users.
Note: Each provider requires appropriate API credentials and configuration. Please refer to the module documentation for setup instructions.
