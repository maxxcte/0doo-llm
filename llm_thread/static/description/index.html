LLM Integration Module

This module provides seamless integration with various Large Language Model (LLM) providers in Odoo.

The LLM Integration module enables direct interaction with AI language models from multiple providers within your Odoo environment. It offers a unified interface for chat completions, text embeddings, and model management across different LLM providers.

Key Features

  • Integration with multiple LLM providers (OpenAI, Anthropic, Ollama, Replicate)
  • Unified chat interface for AI interactions
  • Support for text embeddings and completions
  • Flexible model management and configuration
  • Real-time streaming responses
  • Thread-based conversation history

Supported Providers

OpenAI

  • Chat completions with GPT models
  • Text embeddings
  • Automatic model discovery

Anthropic

  • Integration with Claude models
  • Support for latest Claude-3 models
  • Multimodal capabilities

Ollama

  • Local model deployment
  • Support for various open-source models
  • Custom model configuration

Replicate

  • Access to diverse AI models
  • Support for specialized models
  • Easy model deployment
Note: Each provider requires appropriate API credentials and configuration. Please refer to the module documentation for setup instructions.

Key Components

Chat Interface

The module provides a modern, responsive chat interface featuring:

  • Real-time message streaming
  • Markdown support
  • Message history
  • Export functionality
  • Thread management

Model Management

Comprehensive model management capabilities including:

  • Automatic model discovery
  • Model capability tracking
  • Default model configuration
  • Model-specific parameters

Security

The module implements robust security features:

  • Role-based access control
  • Secure API key management
  • Thread-level privacy
  • Company-specific configurations

Installation

The module requires the following Python packages:

anthropic
ollama
openai
replicate

Configuration

After installation:

  1. Configure LLM providers in the Settings menu
  2. Set up API credentials for each provider
  3. Configure default models for different use cases
  4. Grant appropriate access rights to users
Integration: This module can be extended to add AI capabilities to other Odoo modules through its flexible API.
