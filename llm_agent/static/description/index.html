LLM Agent Module

This module adds agent capabilities to the LLM integration, allowing for specialized AI assistants with dedicated tools and configurations.

The LLM Agent module extends the core LLM integration by providing a framework for creating and managing specialized AI agents. Each agent can have its own configuration, tools, and system prompts, enabling more targeted AI assistance for different use cases.

Key Features

  • ✓ Create and manage specialized AI agents
  • ✓ Assign specific models and providers to each agent
  • ✓ Configure agent-specific tools and capabilities
  • ✓ Easily switch between agents in the chat interface
  • ✓ Customize system prompts for different use cases

Agent Management

The module provides comprehensive agent management capabilities:

  • Create agents with specific names and descriptions
  • Assign LLM providers and models to each agent
  • Configure agent-specific tools and capabilities
  • Set custom system prompts to guide agent behavior
  • Organize agents by categories or departments

Chat Integration

Seamlessly integrate agents into the chat interface:

  • Select agents from a dropdown in the chat header
  • Switch between agents during a conversation
  • Agent settings automatically apply to the conversation
  • Visual indicators show which agent is active
Note: This module requires the base LLM integration module to be installed.

Security

The module implements robust security features:

  • Role-based access control for agent management
  • Regular users can use agents but only managers can create/edit them
  • Company-specific agent configurations
  • Audit logging for agent usage

Use Cases

Create specialized agents for different departments and functions:

  • Customer Support Agent: Trained to answer product questions and troubleshoot issues
  • Sales Assistant: Configured to provide product recommendations and pricing information
  • HR Assistant: Specialized in answering policy questions and onboarding information
  • Technical Expert: Focused on providing technical documentation and code examples

Configuration

After installation:

  1. Navigate to LLM > Configuration > Agents
  2. Create new agents with specific providers and models
  3. Configure tools and system prompts for each agent
  4. Assign agents to specific user groups if needed
Integration: This module extends the core LLM functionality with agent capabilities that can be used across all Odoo modules.
